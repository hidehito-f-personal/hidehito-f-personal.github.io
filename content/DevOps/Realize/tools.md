### DevOpsを実現するツールについて
+ DevOpsを実現するために様々なツールが使われている（DevOps導入指南より）
    + バックログ管理：redmine, JIRA
    + メトリクス収集：zabbix
    + chat（アジャイル開発などで重要）: slack
    + CI/CD: circleCI, jenkins 
    + コード管理：GitLab, GitHub
